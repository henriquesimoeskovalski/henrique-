# calculadora-de-impacto-ambiental-

# 🎯 Objetivo do meu projeto 
calcular o balanço de carbono de uma propriedade rural (subtraindo o que ela emite pelo que ela sequestra da atmosfera) e mostrar visualmente ao produtor se ele tem direito a incentivos financeiros, como juros mais baixos ou venda de créditos de carbono, com base nas suas práticas de conservação.

# 🛠️ Recursos tecnicos do projeto

1. Tecnologias de Estruturação e Estilo (Front-End)
* HTML5: Utilizado para estruturar a interface da calculadora, organizando os formulários de entrada de dados, o painel lateral de diagnóstico e as seções de informações técnicas.

* CSS3 (Variáveis e Flexbox/Grid):

* Uso de CSS Variables (:root) para centralizar a paleta de cores (identidade visual voltada ao agronegócio, com tons de verde e cinza).

* CSS Grid e Flexbox para criar um layout responsivo, que se adapta automaticamente tanto a telas de computadores quanto a celulares e tablets.

* Animações de transição (transition) para preencher as barras do gráfico de forma suave assim que o cálculo é gerado.

2. Lógica de Programação e Dinâmica (JavaScript Vanilla)
* Manipulação do DOM (Document Object Model): O JavaScript intercala a exibição de elementos na tela, ocultando a mensagem inicial de instrução e exibindo o painel de resultados em tempo real.

* Event Listeners: Captura o momento em que o usuário clica no botão de envio (submit) do formulário para processar os dados sem recarregar a página (e.preventDefault()).

* Operadores Modernos (Optional Chaining): Uso do operador ?.value para evitar erros de leitura caso um elemento HTML ainda não tenha sido renderizado na tela.
